# HUB SM - Gestão de Limpeza

Aplicativo interno simples para gestão inicial da rotina de limpeza do HUB SM / Imobiliária Santa Maria.

## Funcionalidades atuais
- Login por senha simples
- Perfil Tezzei administrativo
- Perfis Neia, Selma e Helena
- Pedido Sinval pela Neia
- Produto fora da lista
- Painel Tezzei com pedidos pendentes
- Copiar pedido
- Editar pedido
- Excluir pedido
- Marcar pedido como feito
- Dados salvos temporariamente no localStorage

## Tecnologias
- React
- TypeScript
- Vite
- localStorage

## Observação
Este é um MVP inicial. Ainda não possui backend, banco de dados ou autenticação real.
